# Csharp-game
2D игра написана в рамках учебного проекта по разработке приложений на движке Unity
Реализовано:
- счетчик жизней
- добавление к счетчику сбором бонусных ящиков и урон о кольев
- стрельба
- уничтожение врага
- физика в виде прыжков и анимация их
